## Ringkasan Eksekutif

Jisoi menjual custom pouch DTF dengan differensiasi utama: tanpa minimum order, free mockup, dan proses cepat. Target pelanggan adalah panitia event, tim marketing perusahaan, dan UMKM yang membutuhkan merchandise berlogo. Konversi tidak terjadi di website — **semua lead masuk via WhatsApp**. Strategi Google Ads harus dirancang untuk memaksimalkan klik berkualitas ke WhatsApp, bukan hanya klik ke landing page.

## Fase 5 — Conversion Tracking

### 5.1 Status Tracking Saat Ini

```html
<!-- Google tag (gtag.js) — AW-18080053795 -->
```

### 5.2 Konversi yang Harus Di-tracking

| Event | Cara Track | Nilai (Est.) |
|-------|-----------|--------------|
| **WhatsApp Click** | GA4 event → import ke Ads | Rp50.000 (proxy) |
| **Scroll to Pricing** | GA4 scroll event | Micro conversion |
| **FAQ Click** | GA4 click event | Micro conversion |
| **Time on page > 60s** | GA4 engagement | Micro conversion |

### 5.3 Setup WhatsApp Click Tracking

Tambahkan kode ini di setiap tombol WhatsApp di React component:

```javascript
// Di komponen yang memiliki link WhatsApp
const handleWhatsAppClick = () => {
  // Google Ads conversion
  if (window.gtag) {
    window.gtag('event', 'conversion', {
      send_to: 'AW-18080053795/YOUR_CONVERSION_LABEL',
      value: 50000,
      currency: 'IDR',
    });
  }
  // GA4 event
  if (window.gtag) {
    window.gtag('event', 'whatsapp_click', {
      event_category: 'engagement',
      event_label: 'whatsapp_cta',
    });
  }
};
```

> **Penting:** Buat conversion action baru di Google Ads dashboard:
> - Tipe: Website
> - Kategori: Contact
> - Event Name: `whatsapp_click`
> - Count: Every (setiap klik dihitung)
> - Value: Rp50.000 (estimasi, adjust sesuai rata-rata closing rate)

### 5.4 Import Konversi dari GA4

Di Google Ads > Tools > Conversions > Import from Google Analytics:
- Pilih GA4 property
- Import event `whatsapp_click`
- Set sebagai primary conversion

```
Minggu 1–4: Maximize Clicks

## Fase 8 — Monitoring & Optimasi

### 8.1 KPI Utama

| Metrik | Target | Frekuensi Review |
|--------|--------|-----------------|
| **CTR** | > 5% (Search) | Weekly |
| **CPC** | < Rp3.000 | Weekly |
| **Conversion Rate** | > 3% (klik WA / landing) | Weekly |
| **CPA (per WA click)** | < Rp100.000 | Monthly |
| **Impression Share** | > 40% | Monthly |
| **Quality Score** | > 7/10 | Monthly |

### 8.2 Weekly Optimization Checklist

```
[ ] Review Search Terms Report → tambah negative keywords
[ ] Cek CTR per ad → pause ad dengan CTR < 1%
[ ] Review top keywords → naikkan bid jika conversion rate tinggi
[ ] Cek Quality Score → perbaiki ad relevance jika < 6
[ ] Review device performance → adjust mobile bid
[ ] Cek schedule performance → adjust dayparting
[ ] Update negative keyword list dari irrelevant searches
```

### 8.3 Search Terms Report — Action Rules

| Kondisi | Aksi |
|---------|------|
| Irrelevant search term muncul | Tambah sebagai exact negative keyword |
| High-volume relevant term tidak ada di keyword list | Tambah sebagai phrase match keyword |
| Search term performa bagus (CTR > 8%) | Tambah sebagai exact match keyword terpisah |
| Keyword Quality Score < 5 | Review landing page relevance |

### 8.4 A/B Testing Plan

**Tes 1: CTA Variation (bulan 1)**
- Versi A: "Chat WhatsApp Sekarang"
- Versi B: "Dapatkan Mockup Gratis"
- Ukur: CTR headline

**Tes 2: Value Prop Variation (bulan 2)**
- Versi A: Fokus pada harga (Rp20.000/pcs)
- Versi B: Fokus pada no MOQ (Tanpa Minimum Order)
- Ukur: Conversion rate

**Tes 3: Landing Page Variation (bulan 3)**
- Versi A: Current landing page
- Versi B: Dedicated landing page dengan form singkat
- Ukur: Bounce rate & WA click rate

## Fase 9 — Landing Page Optimization untuk Ads

### 9.1 Google Ads Quality Score Factors

Quality Score (1–10) ditentukan oleh:
1. **Expected CTR** → ad copy harus match keyword
2. **Ad Relevance** → keyword harus ada di headline/description
3. **Landing Page Experience** → kecepatan + relevansi konten

### 9.2 Checklist Landing Page Jisoi

```
[✅] Keyword "custom pouch" ada di H1/title
[✅] CTA WhatsApp jelas dan terlihat above the fold
[✅] Mobile-responsive
[✅] HTTPS aktif
[✅] Harga ditampilkan (penting untuk trust)
[⚠️] Page speed — target < 3 detik (cek Google PageSpeed Insights)
[⚠️] Meta description match dengan ad copy
[❌] Dedicated landing page tanpa navigasi (higher conversion)
```

### 9.3 UTM Parameters untuk Tracking

Tambahkan UTM ke semua URL di Google Ads:

```
https://www.jisoi.net/?utm_source=google&utm_medium=cpc&utm_campaign=custom-pouch-search&utm_content={adgroupid}&utm_term={keyword}
```

Di Google Ads, gunakan ValueTrack parameters:

```
